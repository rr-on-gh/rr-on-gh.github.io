Features

Reading Practice

Learn new German phrases with English translations. Perfect for building vocabulary and understanding sentence structure.

Speaking Practice

Record your German pronunciation and receive instant AI-powered feedback. Practice real conversations and improve your speaking skills.

Multiple Difficulty Levels

Choose from A1 (beginner) to C2 (advanced) based on the Common European Framework of Reference for Languages (CEFR).

  • A1 - Beginner: Basic phrases and simple sentences
  • A2 - Elementary: Common everyday situations
  • B1 - Intermediate: More complex conversations
  • B2 - Upper Intermediate: Detailed discussions
  • C1 - Advanced: Complex language use
  • C2 - Proficient: Near-native fluency

Real-World Scenarios

Practice conversations in 8 different everyday situations:

  • Stores/Shopping
  • Restaurants/Cafes
  • Doctor/Pharmacy
  • Salon/Barber
  • Parties/Social
  • Work/Office
  • Neighbors
  • Transport/Travel

Sentence Sources

Local Database: Practice with 350+ pre-loaded sentences across all levels and scenarios. Works offline.

GenAI: Generate custom sentences using AI based on your preferences and keywords. Requires API key.

iCloud Sync

Your progress automatically syncs across all your iOS devices using iCloud.

Settings

Sentence Source

Choose between Local Database (default) or GenAI for sentence generation.

AI Provider

When using GenAI, select your preferred provider:

  • OpenAI (default) - Requires OpenAI API key
  • Bedrock - Requires AWS Bedrock API key

API Keys

Enter your API keys for AI-powered features. Keys are stored securely on your device.

  • OpenAI API Key: Get from platform.openai.com
  • Bedrock API Key: Bedrock is an AWS service. You need an AWS account and can get your API key from AWS Bedrock documentation

Language Level

Select your current German proficiency level (A1-C2). Default is A2.

Conversation Setting

Choose the scenario you want to practice (stores, restaurants, doctor, etc.).

Keywords (Optional)

When using GenAI, add specific keywords to customize generated sentences to your interests.

How to Use

Reading Practice

  1. Tap Reading Practice on the home screen
  2. Configure your level and setting in Settings
  3. Tap Generate Sentences
  4. Read through the English and German translations
  5. Swipe to see more sentences

Speaking Practice

  1. Tap Speaking Practice on the home screen
  2. Grant microphone and speech recognition permissions when prompted
  3. Tap Get Next Sentence
  4. Read the English sentence
  5. Tap Start Recording and speak the German translation
  6. Tap Stop Recording when done
  7. Tap Check Translation to receive AI feedback
  8. Review your transcription and feedback

Privacy

Practice German respects your privacy:

  • Microphone access is only used for speech recognition during practice
  • Speech recognition is processed on-device when possible
  • API keys are stored securely on your device
  • No personal data is collected or shared
  • iCloud sync is optional and controlled by your device settings

Troubleshooting

Speech Recognition Not Working

  • Check microphone permissions in Settings > Privacy > Microphone
  • Check speech recognition permissions in Settings > Privacy > Speech Recognition
  • Ensure you have a stable internet connection
  • Speak clearly and at a moderate pace

GenAI Not Generating Sentences

  • Verify your API key is entered correctly in Settings
  • Check that you have API credits available
  • Ensure you have an internet connection
  • Try switching to Local Database mode

iCloud Sync Issues

  • Check that iCloud Drive is enabled in device Settings
  • Ensure you're signed in to iCloud
  • Check available iCloud storage
  • Wait a few moments for sync to complete
